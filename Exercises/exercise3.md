## Chess Queens

You are given coordinates of two queens on a chess board. Find out whether or not they hit each other.

### Input data format

Four integer numbers x1, y1, x2, y2

### Output data format

Type "YES" (uppercase) if they hit each other or "NO" if they don't.

### Sample Input 1:

1 1 3 3
### Sample Output 1:

YES
### Sample Input 2:

1 1 4 3
### Sample Output 2:

NO
### Sample Input 3:

2 2 5 2
### Sample Output 3:

YES
